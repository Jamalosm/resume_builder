# Django LaTeX Resume Builder

A Django-based resume builder that:
- Auto-fills data from uploaded resumes
- Supports manual editing
- Generates professional LaTeX-based PDFs
- Auto-formats experience, projects, and skills

## Tech Stack
- Django
- LaTeX (pdflatex / MiKTeX)
- Python

## Features
- Resume upload & auto-detection
- Editable form
- ATS-friendly LaTeX output
- Dynamic bullet & section formatting
